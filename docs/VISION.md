# Scoutly Vision

Scoutly helps people find the best used deal without searching through hundreds of listings.

## Initial niche

Version 1 focuses only on used GPUs.

## Long-term direction

Scoutly can expand into other used product categories such as CPUs, camera gear, game consoles, 3D printers, Lego, and Apple devices.

## Core promise

Search once. Scoutly compares marketplaces and shows the best result from each selected retailer.
