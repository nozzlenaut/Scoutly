# Scoutly v0.6.3 / PriceSift console family hotfix

- Aligns every console-family child search with the same exact catalog product used when that model is selected directly.
- Fixes cases where `All models` could return nothing while selecting Series X returned a listing.
- Rejects console storage accessories including external hard drives, external SSD/HDD listings, storage expansion cards, game drives, and drive-only listings.
- Keeps internal Scoutly infrastructure unchanged.
