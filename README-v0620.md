# PriceSift v0.6.20

## KEH lens finder preview polish

- Removes the exact-model search field from the lens flow.
- Uses the intended guided experience: Mount/System → Prime/Zoom → Focal-length group → optional brand → available models.
- Keeps model results hidden until the three required choices are complete.
- Adds model images, availability counts, lowest price, and KEH grade summaries.
- Expands each model into the three lowest-priced current KEH listings.
- Keeps the entire lens experience private and out of public search, QA, and price history.
