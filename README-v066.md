# Scoutly v0.6.6 / PriceSift console direct-search rollback

- Replaces the Console builder with the same direct autocomplete search used by Cameras and GPUs.
- Keeps RAM as the only specification builder.
- Console autocomplete now exposes exact catalog models instead of requiring brand/family/model steps.
- Normal console searches resolve only static catalog products; they no longer synthesize family-level builder products.
- Preserves original Nintendo Switch aliases, including V1, V2, HAC-001, HAC-001(-01), Standard, and Original Switch wording.
- Keeps the multi-query marketplace coverage for the original Switch revisions and all v0.6.5 filtering improvements.
